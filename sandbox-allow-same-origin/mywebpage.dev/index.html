Simple Webpage to illustrate a Sandboxed IFrame with "allow-same-origin"

Both iframes point to the same target - to an "advertiser".
Both iframes are sandboxed, but this example should show the difference between an sandboxed iframe with the parameter "allow-same-origin" and one without this parameter. The first iframe sandboxed - the advertiser tries to show his origin using JavaScript. The only parameter which is set into the sandbox is "allow-scripts".
The second one is sandboxed - with "allow-same-origin" - the advertiser is now able to show his origin.

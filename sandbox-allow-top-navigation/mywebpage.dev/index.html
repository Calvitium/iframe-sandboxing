Simple Webpage to illustrate a Sandboxed IFrame with "allow-top-navigation"

Both iframes point to the same target - to an "advertiser".
Both iframes are sandboxed, but this example should show the difference between an sandboxed iframe with the parameter "allow-top-navigation" and one without this parameter. The first iframe is sandboxed - the advertiser represents a link which would replace this page (mywebpage.dev) by one of the advertiser (advertiser.dev:9999/secondpage.html) through setting the anchor-attribute "target" to "_top". The Sandbox will block the execution when the link is clicked.
The second one is sandboxed - with "allow-top-navigation" - if the link is clicked in this iframe, this page will be replaced by the links target.

Simple Webpage to illustrate a Sandboxed IFrame

Both iframes point to the same target - to an "Advertiser".
The first iframe is not sandboxed - the advertiser uses JavaScript to change his own DOM (malicous content here).
The second one is sandboxed and the advertiser is no longer allowed to exectute his JavaScript-Code.

This simplest example shows the basic difference between a non-sandboxed iframe and a sandboxed one.
